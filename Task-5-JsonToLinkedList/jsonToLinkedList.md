# Функция, которая преобразует JSON-массив объектов в связный список.

## [Реализация](./index.js)

## Постановка 
- Разработайте функцию преобразования JSON в связный список. На входе функция должна получать JSON, содержащий список объектов, на выходе объект, представляющий из себя односвязный список.

## Реализованные функции
- `jsonToLinkedList(json)`

### Параметры

- `json`: Массив объектов, представляющих данные для связного списка.

Возвращает `ListNode | null`: Первый узел связного списка или null, если входной массив пуст.
### Пример использования

```javascript
  const collectionObject = [
    { value: 1 },
    { value: 2 },
    { value: 3 }
    ];
  
  const linkedList = jsonToLinkedList(collectionObject);
  console.log(linkedList);
```
